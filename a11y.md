
# ARIA Cheatsheet

## aria-label
Задает текстовое описание элемента, если видимого текста нет.

Используется для:
- кнопок с иконками;
- SVG;
- input без label.

```
<button aria-label="Закрыть">
  ❌
</button>
```

---

## aria-labelledby
Берет название элемента из другого элемента по `id`.

Используется для:
- modal;
- dialog;
- form;
- section.

```tsx
<h2 id="title">
  Настройки
</h2>

<div
  role="dialog"
  aria-labelledby="title"
>
```

Screen reader озвучит:

```
Настройки
```

---

## aria-describedby
Добавляет дополнительное описание элемента.

Используется для:
- подсказок;
- сообщений об ошибках;
- инструкций.

```tsx
<p id="help">
  Введите пароль минимум из 8 символов.
</p>

<input
  aria-describedby="help"
/>
```

Screen reader сначала прочитает название поля, затем описание.

---

## aria-hidden
Скрывает элемент от screen reader.

Используется для:
- декоративных иконок;
- эмодзи;
- элементов, которые не несут смысловой нагрузки.

```tsx
<span aria-hidden="true">
  ❤️
</span>
```

---

## aria-expanded
Показывает, открыт элемент или закрыт.

Используется для:
- Accordion;
- Dropdown;
- Menu;
- Popover.

```tsx
<button
  aria-expanded={isOpen}
>
  Меню
</button>
```

Если `isOpen === true`

```
expanded
```

Если `false`

```
collapsed
```

---

## aria-controls
Связывает элемент управления с элементом, которым он управляет.

Используется для:
- Dropdown;
- Accordion;
- Tabs;
- Popover.

```tsx
<button
  aria-controls="menu"
>
  Меню
</button>

<ul id="menu">
  ...
</ul>
```

---

## aria-haspopup
Сообщает, что элемент открывает всплывающее окно.

Частые значения:
- menu
- dialog
- listbox
- tree
- grid

```tsx
<button
  aria-haspopup="menu"
>
  Открыть меню
</button>
```

---

## aria-pressed
Используется для переключаемых кнопок (Toggle Button).

```tsx
<button
  aria-pressed={liked}
>
  ❤️
</button>
```

Screen reader:

```
pressed
```

или

```
not pressed
```

---

## aria-selected
Показывает, выбран ли элемент.

Используется для:
- Tabs;
- Listbox;
- Option.

```tsx
<button
  role="tab"
  aria-selected={true}
>
  Главная
</button>
```

---

## aria-checked
Показывает состояние Checkbox, Radio или Switch.

```tsx
<div
  role="checkbox"
  aria-checked={true}
>
</div>
```

---

## aria-disabled
Сообщает, что элемент недоступен.

> Не отключает элемент, а только сообщает об этом screen reader.

```tsx
<button
  aria-disabled="true"
>
  Отправить
</button>
```

---

## aria-current
Показывает текущий активный элемент.

Часто используется для:
- текущей страницы;
- текущего шага;
- текущей даты.

```tsx
<a
  href="/"
  aria-current="page"
>
  Главная
</a>
```

---

## aria-live
Озвучивает изменения автоматически.

Значения:
- polite — дождаться окончания текущего озвучивания;
- assertive — озвучить сразу.

```tsx
<div aria-live="polite">
  Пароль сохранён
</div>
```

---

## aria-busy
Сообщает, что элемент сейчас обновляется или загружается.

```tsx
<div aria-busy="true">
  Загрузка...
</div>
```

---

## aria-invalid
Сообщает, что значение поля некорректно.

```tsx
<input
  aria-invalid="true"
/>
```

---

## aria-required
Сообщает, что поле обязательно для заполнения.

```tsx
<input
  aria-required="true"
/>
```

---

## aria-readonly
Сообщает, что значение можно только читать.

```tsx
<input
  aria-readonly="true"
/>
```

---

## aria-modal
Указывает, что открыт модальный диалог.

Используется вместе с `role="dialog"`.

```tsx
<div
  role="dialog"
  aria-modal="true"
>
</div>
```

---

## aria-orientation
Указывает направление компонента.

Значения:
- horizontal
- vertical

```tsx
<div
  role="tablist"
  aria-orientation="horizontal"
>
</div>
```

---

## aria-valuemin
Минимальное значение.

```tsx
<div
  role="slider"
  aria-valuemin={0}
/>
```

---

## aria-valuemax
Максимальное значение.

```tsx
<div
  role="slider"
  aria-valuemax={100}
/>
```

---

## aria-valuenow
Текущее значение.

```tsx
<div
  role="slider"
  aria-valuenow={45}
/>
```

---

## aria-valuetext
Человекочитаемое значение.

```tsx
<div
  role="slider"
  aria-valuetext="Средняя громкость"
/>
```

---

## aria-autocomplete
Используется для автодополнения.

Значения:
- inline
- list
- both
- none

```tsx
<input
  aria-autocomplete="list"
/>
```

---

## aria-multiselectable
Разрешает выбирать несколько элементов.

```tsx
<div
  role="listbox"
  aria-multiselectable="true"
>
</div>
```

---

## aria-multiline
Показывает, что поле поддерживает несколько строк.

```tsx
<div
  role="textbox"
  aria-multiline="true"
>
</div>
```

---

## aria-activedescendant
Указывает активный элемент внутри списка.

Используется в:
- Combobox;
- Listbox;
- Autocomplete.

```tsx
<input
  aria-activedescendant="option-2"
/>
```

---

## aria-level
Указывает уровень вложенности элемента.

```tsx
<div
  role="treeitem"
  aria-level={3}
>
</div>
```

---

## aria-posinset
Позиция элемента внутри группы.

```tsx
<div
  role="treeitem"
  aria-posinset={2}
>
</div>
```

---

## aria-setsize
Общее количество элементов в группе.

```tsx
<div
  role="treeitem"
  aria-setsize={15}
>
</div>
```
